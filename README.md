

 📰 Web Portal Berita





 🚀 Fitur Utama
- Manajemen Konten Berita: Publikasi artikel berita secara *real-time*.
-Penyimpanan Media Skalabel: Optimasi gambar dan aset visual berkecepatan tinggi.
-Desain Responsif: Tampilan yang nyaman diakses melalui perangkat mobile, tablet, maupun desktop.
-Infrastruktur Cloud Handal: Kombinasi performa server backend/database yang stabil dan *deployment* cepat.



 🛠️ Teknologi yang Digunakan

Proyek ini dikembangkan dan dikelola menggunakan ekosistem teknologi berikut:

 1. Development & Version Control
 -Visual Studio Code (VS Code): Editor teks utama yang digunakan selama proses *coding* dan pengembangan.
 -GitHub: Platform repositori untuk manajemen *source code*, kolaborasi tim, dan pelacakan versi (*version control*).

 2. Backend, Database & Storage Cloud
-Firebase Storage: Digunakan untuk penyimpanan data digital dan manajemen dokumen secara aman.
-Cloudinary: Solusi berbasis cloud khusus untuk manajemen media (optimasi, kompresi, dan pengiriman gambar/video berita) agar *loading* halaman lebih cepat.
-Hostinger: Digunakan sebagai penyedia layanan hosting utama/server backend dan manajemen domain web berita.

 3. Deployment & Hosting Front-End
-Vercel: Platform cloud yang digunakan untuk melakukan *deployment* cepat dan *hosting* aplikasi front-end secara otomatis yang terintegrasi dengan GitHub.



 📦 Langkah Instalasi (Lokal)

Jika Anda ingin menjalankan proyek ini di lingkungan lokal Anda, ikuti langkah-langkah berikut:

1. Clone Repositori:
   ```bash
   git clone [https://github.com/username/nama-repositori-anda.git](https://github.com/username/nama-repositori-anda.git)



2. Masuk ke Direktori Proyek:
```bash
cd nama-repositori-anda

```


3. Instalasi Dependensi:
(Sesuaikan dengan package manager yang Anda gunakan, contoh: npm)
```bash
npm install

```


4. Konfigurasi Environment Variables (`.env`):
Buat file `.env` di akar proyek dan masukkan kredensial dari **Firebase**, **Cloudinary**, dan **Hostinger** Anda.
5. Jalankan Aplikasi:
```bash
npm run dev

```
